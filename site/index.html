This piece is an experiment in creating an a split-flap or LED like display using only HTML. There is no JavaScript, CSS, or images involved. The display is built by using deprecated 'marquee' tags inside table cells. Each tag has a pattern of letters and non-breaking spaces to coordinate the display of the letters.
